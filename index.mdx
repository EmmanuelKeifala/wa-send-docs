# Ship WhatsApp messaging that feels product-ready from day one

wa-send helps teams deliver verification codes, alerts, reminders, and account updates over WhatsApp with a cleaner integration surface and a more polished delivery experience.

Whether you are replacing fragile SMS flows or adding a higher-trust notification channel, wa-send keeps the path from first API call to production rollout short and predictable.

- recovery, security, and account alerts
- operational notifications for internal products and tools

## A fast path to first send

1. Create an account and generate an API key.
2. Send a direct message or a template-based request.
3. Move into production using the same API shape.

## Developer workflow

1. Use bearer auth to create templates and issue keys.
2. Use API keys for backend message sends.
3. Treat `202 Accepted` as queued for processing.
4. Handle `400`, `401`, `404`, and `500` responses explicitly.
