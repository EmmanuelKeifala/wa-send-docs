This guide shows how to call wa-send from Python using the `requests` library.

## On this page

- install requests
- configure your client
- send direct messages
- send template messages
- create templates with bearer auth
- production tips

## Install requests

```bash
pip install requests
```

## Configure your client

```python
import os
import requests

API_BASE_URL = os.getenv("WASEND_BASE_URL", "https://api.wa-send.dev")
API_KEY = os.getenv("WASEND_API_KEY")


def send_request(path, payload):
    response = requests.post(
        f"{API_BASE_URL}{path}",
        headers={
            "Content-Type": "application/json",
            "x-api-key": API_KEY,
        },
        json=payload,
        timeout=30,
    )

    data = response.json()

    if not response.ok:
        raise RuntimeError(data.get("error", "Request failed"))

    return data
```

## Send a direct message

```python
result = send_request(
    "/api/messages",
    {
        "to": "+23276123456",
        "message": "Your verification code is 483920. It expires in 5 minutes. Do not share this code.",
    },
)

print(result)
```

## Send with a template

Templates are the recommended option for recurring product flows.

```python
result = send_request(
    "/api/messages",
    {
        "to": "+23276123456",
        "templateId": "template-verification-id",
        "templateArgs": {
            "otp": "483920",
            "expires": "5",
        },
    },
)

print(result)
```

## Create a template

```python
def create_template(jwt_token):
    response = requests.post(
        f"{API_BASE_URL}/api/templates",
        headers={
            "Content-Type": "application/json",
            "Authorization": f"Bearer {jwt_token}",
        },
        json={
            "name": "Order Update",
            "template_type": "custom",
            "message_template": "Hi {{customer_name}}, your order {{order_id}} is on the way.",
            "otp_length": 6,
            "expires_in": 300,
        },
        timeout=30,
    )

    data = response.json()

    if not response.ok:
        raise RuntimeError(data.get("error", "Could not create template"))

    return data
```

## Production tips

- store API keys in environment variables or a secrets manager
- set explicit request timeouts
- use templates for repeated workflows
- keep `test` and `live` credentials separate
- treat `202` as queued, not final delivery confirmation
